# Taller 6 - Restaurantes (Swing + Java2D)

## Instrucciones para ejecutar

1. Abrir Eclipse
2. File → Import → General → Existing Projects into Workspace
3. Seleccionar la carpeta del proyecto
4. Click en Finish
5. En el explorador de archivos de Eclipse, abrir la clase
   `uniandes.dpoo.swing.interfaz.principal.VentanaPrincipal`
6. Click derecho → Run As → Java Application

## Funcionalidades

- La ventana principal muestra la lista de restaurantes y los detalles
  del restaurante seleccionado (nombre, calificación en estrellas, visitado)
- Botón "Nuevo": abre una ventana para agregar un restaurante,
  con mapa interactivo para marcar la ubicación con click
- Botón "Ver todos": abre el mapa con todos los restaurantes marcados
- En la ventana del mapa se puede filtrar entre "Todos" y "Visitados"
  usando los radio buttons
